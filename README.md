# Creative Nex Landing Page

A modern, responsive landing page for Creative Nex - Web Application, Desktop Application, and Product Development Company.

## Features

✨ **Modern Design**
- Light color scheme with gradient accents
- Smooth animations and transitions
- Fully responsive (Mobile, Tablet, Desktop)

🚀 **Sections**
- Navigation Header with smooth scrolling
- Hero Section with CTA buttons
- Services Section with 4 main services and development flow
- About Section with company information
- Contact Section with integrated form and contact details
- Footer with social links

💻 **Technology Stack**
- React.js 18.2.0
- CSS3 with Animations
- React Icons
- React Scroll for smooth navigation

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Navigate to the website directory:
```bash
cd c:\Users\BaranidharanThiruven\Pictures\website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

### Contact Information
Update the contact details in the following files:
- `src/components/Contact.js` - Contact form and info cards
- `src/components/Navigation.js` - Navigation button email
- `src/components/Footer.js` - Footer contact information

### Colors
The color scheme uses a gradient of:
- Primary: `#6c63ff` (Purple)
- Secondary: `#a29bfe` (Light Purple)
- Background: `#ffffff` (White)

Modify colors in the CSS files to match your preferences.

### Services
Update the services list in `src/components/Services.js` to customize the service offerings.
